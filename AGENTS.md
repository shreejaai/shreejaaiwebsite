# AGENTS.md - Working Agreement

## Coding style
- Keep HTML semantic and accessible.
- Include `aria-label` (and related ARIA attributes where useful), especially for interactive controls like mobile navigation.
- Keep all CSS in `assets/style.css` and avoid duplication.
- Keep JavaScript minimal and focused on progressive enhancement.

## Delivery checklist
- Open each page manually in a browser and verify layout consistency.
- Validate internal links and CTA buttons.
- Confirm sticky navbar behavior and mobile hamburger toggle.
- Confirm contact details and mailto interactions are correct.
